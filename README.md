# ChatbotJS

Añade un chatbot a tu web con dos lineas de código

```HTML

<link rel="stylesheet" href="https://cdn.devetty.es/ChatbotJS/css">
<script type="module" src="https://cdn.devetty.es/ChatbotJS/js"></script>

```
